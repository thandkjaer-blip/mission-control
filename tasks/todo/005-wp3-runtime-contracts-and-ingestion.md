# 005 - WP3 Runtime contracts and ingestion

## Owner
Developer / Product Owner

## Goal
Definere og implementere MVP-kontrakter for heartbeats, state updates, events, logs, metrics og cost records.

## Scope
- Definér payload schemas
- Implementér ingestion handlers eller consumers
- Tilføj validation og correlation IDs
- Implementér reconciliation for core runtime state

## Done when
- Runtime-kontrakter er nedskrevet og brugbare
- Ingestion-flow eksisterer for MVP-data
- Core state kan holdes konsistent
