# SCOPE

## In scope for MVP
- Agentliste og agentdetaljer
- Taskliste og workflowliste
- Simpel dependency-view
- Logs, events og metrics views
- Cost/usage overblik
- Alerts view
- Auditérbar command pipeline
- Basis governance/RBAC

## Out of scope for første iteration
- Avanceret auto-remediation
- Predictive anomaly detection
- Multi-tenant support
- Kompleks policy engine
- Fuld enterprise workflow editor

## Arbejdsregel
Vi bygger først noget robust og brugbart, ikke et oppustet kontrolpanel.
